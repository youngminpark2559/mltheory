================================================================================ Covariance matrix is composed of variance and covariance Variance values are located in diagonal axis Covariance values are located in off-diagonal regions ================================================================================ Characteristics of covariance - If $$$x_i$$$ and $$$x_k$$$ increase together, covariance value $$$c_{ik}>0$$$ - If $$$x_k$$$ increases and $$$x_i$$$ decreases, covariance value $$$c_{ik}\lt 0$$$ - If $$$x_i$$$ and $$$x_k$$$ are uncorrelated, covariance value $$$c_{ik} = 0$$$ - $$$|c_{ik}| \lt \sigma_i\times \sigma_k$$$, $$$\sigma_i$$$ is standard deviation of $$$x_i$$$ - $$$c_{ii} = \sigma_i^2 = VAR(x_i)$$$ ================================================================================ /home/young/Pictures/2019_04_07_22:16:53.png $$$\rho_{ik}$$$: correlation coefficient $$$c_{ik} = \rho_{ik} \sigma_i\sigma_k$$$ $$$\rho_{ik}=\dfrac{c_{ik}}{\sigma_i\sigma_k}$$$ ffff
