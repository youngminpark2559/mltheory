12-02 Dimentionality reduction by PCA(principle component analysis)

@
$\begin{bmatrix} x_{1}\\x_{2}\\...\\x_{N} \end{bmatrix} \underrightarrow{linear features extraction} \begin{bmatrix} y_{1}\\y_{2}\\...\\y_{M} \end{bmatrix}=
\begin{bmatrix} w_{11}&w_{11}&...&w_{1N}\\w_{21}&w_{22}&...&w_{2N}\\...\\w_{M1}&w_{M2}&...&w_{MN} \end{bmatrix}\begin{bmatrix} x_{1}\\x_{2}\\...\\x_{N} \end{bmatrix}$

M*N $\cdot$ N*1 $\Rightarrow$ M*1, where N$>$M

@
feature extraction 을 할때 mapping function y=f(x) 의 선택은 우리가 optimization 시키고자 하는 목적에 의해서 결정된다. 크게 두가지 방법이있다
1. 신호표현 signal representation
feature extraction 의 목적이 extraction 에 의해서 dimensionality 가 낮아지더라도 낮은 dimensinal space 에서 원래 data 의 정보들이 손실되지 않고 정확하게 표현하고자할때, 선형변환의 방법으로 PCA 를 사용한다. PCA 를 사용하면 낮은 차원에서도 feature vector 의 feature 가 정확히 표현된다
1. classification
feature extraction 을 할 때, extraction 의 목적이 낮은 차원에서 classification 을 더 잘 하길 원할때, 선형판별분석법 LDA linear discriminant analysis 를 사용한다
