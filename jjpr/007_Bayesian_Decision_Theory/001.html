When you use "discrete random variable" for cumulitive distribution function, derivative of cumulitive distribution function becomes "probability mass function". When you use "continuous random variable" for cumulitive distribution function, derivative of cumulitive distribution function becomes "probability density function". ================================================================================ When you deal with continuous random variable in your statistical project, how can you give values to other people? For example, will you give all continuous x values and its y probability values like this? x=-1 -> probability value 0.3 x=0 -> probability value 0.2 x=1 -> probability value 0.3 ...? Giving infinite continuous x values and its y probability values manually is not possible ================================================================================ That's the reason why you need a formula. ================================================================================ Functions which represent "probability density" can have various and arbitrary shapes. But practically, many cases have pattern-based probability distribution like this ================================================================================ How to express those graphs into a formula? Formulas which can represent above graphs are created by mathematicians. ================================================================================ This is "univariate Gaussian probability density function" $$$f_X(X)=\frac{1}{\sqrt{2\pi}\sigma}\exp \left[ -\frac{1}{2} \left( \frac{X-\mu}{\sigma} \right)^2 \right]$$$ $$$f_X(X)$$$: univariate X, function f changes wrt X $$$\sigma$$$: standard deviation ($$$\sigma^2$$$ is variance) $$$\mu$$$: expectation value, average $$$e$$$: 2.146... ================================================================================ Simple notation for Gaussian probability density function $$$f(x;\mu=0,\sigma^2=4)$$$ Formula notation for Gaussian probability density function $$$\frac{1}{\sqrt{2\pi}2}\exp \left[ -\frac{1}{2} \left( \frac{X-0}{2} \right)^2 \right]$$$ ================================================================================ "Univariate Gaussian probability density function" has one variable X like this See x axis. It can have one value at a time like $$$x=-5, -4, \cdots$$$ ================================================================================ "Multivariate Gaussian probability density function" $$$f_X(X)=\frac{1}{(2\pi)^{\frac{n}{2}}\sqrt{|\Sigma|}} \exp \left[ -\frac{1}{2} (X-\mu)^T\Sigma^{-1}(X-\mu) \right]$$$ $$$n$$$: dimension of vectors ================================================================================ $$$X=\begin{bmatrix} x_1\\x_2\\\vdots\\x_N \end{bmatrix}$$$ If $$$n=2$$$, X can have 2 variables like height=170, weight=60 at a time. ================================================================================ $$$\mu=\begin{bmatrix} \mu_1\\\mu_2\\\vdots\\\mu_N \end{bmatrix}$$$ $$$\mu_1$$$: Average from random variable 1 $$$\vdots$$$ $$$\mu_N$$$: Average from random variable N If $$$n=2$$$ like height and weight, and suppose you extract height and weight data from 10 people. $$$\mu_1$$$: average of 10 heights $$$\mu_2$$$: average of 10 weights ================================================================================ When you use univariate ramdom variable, you use variance (actually you use std $$$\sigma$$$ from $$$\sqrt{\sigma^2}$$$). But when you use multivariate ramdom variable, you use covariance $$$\Sigma$$$. $$$\Sigma=\begin{bmatrix} \sigma_1^2 & & & \\ & \sigma_2^2 & & \\ & & \ddots & \\ & & & \sigma_N^2 \end{bmatrix}$$$ $$$\sigma_1^2$$$: variance of random variable 1 $$$\vdots$$$ $$$\sigma_N^2$$$: variance of random variable N Off-diagonal regions: covariance values ================================================================================ When $$$n=2$$$, for example, when you have 2 random variables like random_variable1=body_weight, random_variable2=body_height multivariate Gaussian probability distribution function shapes like this ================================================================================ Simple notation $$$f(\begin{bmatrix} x_1\\x_2 \end{bmatrix};\mu=\begin{bmatrix} 1\\2 \end{bmatrix},\Sigma=\begin{bmatrix} 2&&0\\0&&4 \end{bmatrix})$$$ Formula notation $$$\frac{1}{\sqrt{(2\pi)^2}\sqrt{8}}\exp \left[ -\frac{1}{2} \begin{bmatrix} x_1-1&&x_2-2 \end{bmatrix} \begin{bmatrix} 0.5&&0\\0&&0.25 \end{bmatrix} \begin{bmatrix} x_1-1\\x_2-2 \end{bmatrix} \right]$$$
